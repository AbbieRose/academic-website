Overview


Machine learning is more powerful than ever, and it is increasingly present and influential in daily life. ML has vast potential as a tool, distinct from programming, that people can use to make creative, personally meaningful projects. This potential is currently underrealized, so we built AlpacaML to explore how beginners might use an ML modeling tool.

AlpacaML is an iOS application that connects to wearable sensors. It allows people to model physical movement using a Dynamic Time Warping-based 1NN classifier. We have used this tool to investigate how youth with no programming experience can incorporate ML classifiers into athletic practice by building models of their own physical activity. We also built an AlpacaML Scratch extension so children could use AlpacaML models of gesture to controle their Scratch projects.
